Model Context Protocol (MCP) Server

Welcome to the Dr. Claude MCP Server. This server provides AI code generation, context storage, and model metrics.

Available Endpoints

/claude-code-generate

Generate code using Dr. Claude's intelligence engine.

POST /claude-code-generate
Content-Type: application/json

{
  "task": "Create a function that calculates factorial",
  "language": "javascript"
}

/context-storage

Store and retrieve conversation context.

GET /context-storage
POST /context-storage
Content-Type: application/json

{
  "context": "Your context data here",
  "sessionId": "unique-session-id"
}

/model-metrics

Get real-time metrics about model performance.

GET /model-metrics

For more information, see the API documentation.
